# Travel Guide Site

Static travel guide site starter.

## Files

- `index.html`: main page
- `styles.css`: page styles
- `app.js`: route data and rendering

## Purpose

This project is set up so itinerary content can be swapped quickly while keeping:

- a polished single-page layout
- a city-to-city route diagram
- timeline and travel notes

## Next steps

1. Replace the sample route data in `app.js`.
2. Update the narrative sections in `index.html`.
3. Preview locally with any static server.
4. Publish through GitHub Pages or Netlify.
